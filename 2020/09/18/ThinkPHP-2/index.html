三、原生查询

  1. 使用 query()方法，进行原生 SQL 查询，适用于读取操作，SQL 错误返回 false； Db::query(‘select * from tp_user’);
  2. 使用 execute 方法，进行原生 SQL 更新写入等，SQL 错误返回 false； Db::execute(‘update tp_user set username=”孙悟空” where id=29’);
